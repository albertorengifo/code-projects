# Product Scope

## Purpose

Define the initial product boundary for the todo list experiment.

## Current Scope

- a web-based todo list experience
- a frontend client for task management
- a backend API for future task persistence and business rules
- a shared package for contracts, schemas, and reusable types
- create, edit, complete, and delete flows for individual tasks

## Out Of Scope For This Phase

- authentication
- multi-user collaboration
- production infrastructure
- advanced analytics

## Notes

Keep the first product slice intentionally narrow so Codex can build it in small increments.
